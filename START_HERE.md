# Celentrail — ONE LINK

## 1) Что делать любому ИИ (без вопросов)
Открой:
- CANONICAL.md (принцип работы)
- spec/celentrail.yaml (ВСЕ размеры/материалы/толщины)
Менять можно ТОЛЬКО spec/celentrail.yaml.
Сгенерированные файлы не трогать руками.

## 2) Актуальные артефакты (обновляются автоматически Actions)
- 3D: models/latest.glb
- CAD for CAE: cad/latest.step
- Отчёт: reports/latest.md

## 3) 3D просмотр (PlayCanvas)
1) Открой models/latest.glb
2) Нажми Raw и скопируй ссылку
3) Вставь в https://playcanvas.com/model-viewer/ → Load model from URL
