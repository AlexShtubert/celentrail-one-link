# Celentrail — ONE LINK / ZERO QUESTIONS

## 1) Что делать любому ИИ (без вопросов)
Открой:
- `CANONICAL.md` — принцип работы
- `spec/celentrail.yaml` — ВСЕ размеры/материалы/ограничения

**Правило:** менять можно ТОЛЬКО `spec/celentrail.yaml`.  
Сгенерированные файлы руками не трогать.

## 2) Актуальные артефакты (обновляются автоматически через GitHub Actions)
- 3D (GLB): `models/latest.glb`
- CAD for CAE (STEP): `cad/latest.step`
- Отчёт: `reports/latest.md`

## 3) 3D просмотр на телефоне (PlayCanvas)
1) Открой файл `models/latest.glb` в GitHub
2) Нажми **Raw** и скопируй прямую ссылку
3) Вставь её в PlayCanvas Model Viewer: https://playcanvas.com/model-viewer/ → **Load model from URL**

## 4) Важные ограничения
- Геометрию/конструкцию нельзя менять без явного подтверждения владельца.
- Пути `models/latest.glb`, `cad/latest.step`, `reports/latest.md` нельзя переименовывать.
