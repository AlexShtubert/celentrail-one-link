# Celentrail — CANONICAL

## Цель
Ты задаёшь компоновку/толщины/материалы через spec/celentrail.yaml.
Дальше автоматически обновляются:
- models/latest.glb (визуальный контроль)
- cad/latest.step (для сопромата)
- reports/latest.md (кинематика/нагрузки)

## Принцип работы (кратко)
Консольная скрытая подвесная система:
- верхние ролики несущие (главные)
- нижние ролики — прижим/стабилизация
Контакт по V-профилю: V на ролике и ответная V на дорожке.

## Расчётная база (по умолчанию)
Базовые числа лежат в spec/celentrail.yaml (их меняем там).
Отчёт с вычислениями: reports/latest.md

## Правило изменений (важно)
Геометрию меняем только по прямой команде владельца.
ИИ меняет только spec/celentrail.yaml и (если нужно) tools/* по инструкции владельца.
